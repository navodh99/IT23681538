# SwiftTranslator Automation Tests

This repository contains automated UI test scenarios for  
https://www.swifttranslator.com using **Playwright with TypeScript**.

---

## Prerequisites

Make sure the following are installed on your machine:

- **Node.js** (v22.17.0 or later)
- **Git** (version 2.50.1.windows.1)

To verify:
```bash
node -v
git --version


Installation

Clone the repository:

git clone https://github.com/<navodh99>/IT23681538.git


Navigate into the project folder:

cd IT23681538


Install dependencies:

npm install


Install Playwright browsers:

npx playwright install



Run All Tests

To execute all automated test scenarios:

Right click Tests folder > Run tests

View Test Report

After execution, open the HTML report using:

npx playwright show-report
